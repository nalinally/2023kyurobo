## ３．遊ぶ

### スタート方法

両者がスイッチを押すとカウントが始まり、その状態で3秒間押し続けて、真ん中のLEDが光るとゲームスタートです

### ゲーム中

ひたすらスイッチを連打します。最初は真ん中のLEDが光っていますが、より連打数が多いほうに徐々に引き寄せられていきます。

### ゲームの勝敗

自分に一番近いLEDを先に光らせたほうの勝利です。
